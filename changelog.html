Details regarding the master branch of Plex Requests, specific releases can be downloaded here or at links below.


v1.1.0

Released: 2015/11/04

  • Allowing banning of users from logging in

v1.0.0

Released: 2015/11/04

  • Entirely new code base
  • Approval system for requests
  • Auto-downloading of TV episodes if all episodes requested
  • Download status provided for TV Shows now
  • New visual style more in line with Plex
  • Numerous other quality of life and behind the scenes changes

v0.5.1

Released: 2015/05/23

  • Version information available at /about that doesn't rely on Git commits
  • TV requests now ask whether you want new episodes, or all episodes (@drzoidberg33)
  • Branch created for those with multiple CP servers

v0.5

Released: 2015/05/19

  • Switch search providers to TheMovieDB
  • Improved search to provide instant results as you type (@jeradin)
  • TV Series support added
  • SickRage and Sonarr integration for automatic TV Series downloads
  • Branch and Commit information added at /about
  • New test pages created and updated (@jrudio)

v0.4

Released: 2015/04/16

  • Added Plex Authentication, a simple way to limit access to Plex friends and managed users
  • Plex Auth status page added, Couch Potato updated for easier readability
  • IMDB links now go through www.dereferer.org
  • Admin interface provides links to two status pages and project website

v0.3.2

Released: 2015/04/5

  • Added filtering to requested movies list
  • Added logic, for requested and downloaded movies
  • Added a short FAQ
  • Removed overflow scrolling of requested movies

v0.3.1

Released: 2015/04/4

  • Added CP test page to help with connectivity testing

v0.3

Released: 2015/04/2

  • Updated to Meteor 1.1
  • Added admin interface
  • Application settings are set via admin interface
  • Removed auto-publish

v0.2

Released: 2015/03/29

  • Updated with proper couch potato integration

v0.1

Released: 2015/03/17

  • Initial release of Plex Requests
