Are people constantly asking you to add new content to your Plex Server? Frequent late night texts asking for a new release that's not out yet? Wouldn't it be easier to just send them to a website where they could submit their requests? Enter Plex Requests!


The new features below are for version 1.0 which is coming out very soon!

Features

  • Users can easily search the TheMovieDB for content to request
  • Handy list of requested movies and TV series and basic issue reporting New
  • Simple user authentication with new approval system New
  • Couch Potato integration for automatic downloads of movies
  • SickRage and Sonarr integrationed for automatic TV Series downloads
  • Pushbullet or Pushover notifications to keep up to date with requests

Download and Installation

Head over to GitHub to fork the latest version or visit the releases page to grab a specific release. Installation instructions are available in the README document. Currently all platforms that Meteor runs on are supported, more information about Meteor can be found here.

For those looking to run on Window's, I've written a blog post on how to use Git to install, as well as update, Plex Requests.

For production use, or for those wanting something more stable, stick to the master branch and the releases page. If you want to test new features, or like living a bit on the edge, the dev channel is the place to be!

Looking for Support?

If you're having any problems or need help, please open a GitHub issue or post on the Plex forums. Find a bug or serious issue? Let me know!

Want to help out?

Want to make Plex Requests more awesome? Feel free to fork the repo and submit a pull request! Not a developer or rather not get your hands dirty? You can donate via PayPal to keep things going, or just simply to say thanks!
